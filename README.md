# ЛАБОРАТОРНАЯ РАБОТА №3. Простое веб-приложение. Верстка

**Задача:** Знакомство с node, npm. Верстка интерфейса с карточками (страница списка с фильтрацией и страница подробнее), данные получать через mock объекты (коллекция). Добавить кнопку добавления (копировать первую карточку), кнопку удаления карточки. В хедере на обеих страницах должна быть кнопка «Домой»

**Сделано:**
- Реализована структура приложения, разделенная на компоненты и страницы, что обеспечило высокую степень переиспользуемости кода и удобство управления интерфейсом
- Создано двухстраничное веб-приложение: главная страница с динамическим списком файлов и страница детального просмотра для отображения контента конкретного документа
- Внедрена логика для манипуляции коллекцией документов, включающая методы удаления карточек, добавления новых записей (через копирование) и навигацию между представлениями
- Реализован механизм фильтрации данных в реальном времени, обеспечивающий поиск документов по названию и отбор файлов по их расширению через функцию getFilteredData
- Организована логика отрисовки интерфейса с использованием специализированных компонентов (например, DocFilterbarComponent, DocumentHeaderComponent), что позволило автоматизировать процесс перерисовки страницы при изменении состояния данных

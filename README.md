# ЛАБОРАТОРНАЯ РАБОТА №2 (Калькулятор HTML/CSS/JS)

**Задача:** Создание калькулятора. Функции на JavaScript.

**Сделано:**
- Реализован механизм обработки пользовательского ввода и создана базовая вычислительная архитектура веб-калькулятора
- Вычислительная логика разделена на изолированные модули: написана отдельная функция для обработки классических бинарных операций и специализированная функция-маршрутизатор applyDataTransformation для унарных вычислений
- Запрограммированы базовые арифметические действия (включая накапливаемые операции), а также расширенные унарные функции (квадратный корень, возведение в квадрат, факториал) с использованием механизма передачи стрелочных функций в качестве аргументов (коллбеков)
- Внедрена специализированная операция «Проверить права», алгоритм которой декодирует введенное десятичное число в пятибитную маску и с помощью операции побитового «И» (&) динамически формирует массив доступных привилегий пользователя
- Реализована JavaScript-логика для обработки событий пользовательского интерфейса, в частности — функция динамической смены визуальной темы калькулятора через взаимодействие с выпадающим списком

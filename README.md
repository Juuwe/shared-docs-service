# ЛАБОРАТОРНАЯ РАБОТА №5 (XHR запросы)

**Задача:** добавить страницу добавления/редактирования и соответствующие кнопки, подключение к созданному API бэкенду. Запросы XHR, Cors обойти через расширение браузера CORS Unblock.

**Сделано:**
- Внедрен изолированный архитектурный модуль, включающий класс-маршрутизатор для управления эндпоинтами и сервисный класс Ajax, который реализует выполнение асинхронных CRUD-запросов (GET, POST, PATCH, DELETE) через XMLHttpRequest без перезагрузки страницы.
- Статическая инициализация главной страницы заменена на динамическую, при которой UI-компоненты итеративно генерируются на основе актуального массива данных, получаемого от серверного API.
- Разработан обновленный пользовательский интерфейс для добавления и редактирования карточек, включающий новые поля форм (в том числе для изображений превью) и механизм автоматического предзаполнения данных при редактировании.
- Настроена связь клиентского приложения с локальным бэкендом с применением браузерного расширения CORS Unblock для преодоления ограничений политики безопасности браузера (CORS).

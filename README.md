# Реализация собственного API на Node.js

**Задача:** Реализация собственного API на Node.js. Тестирование через Postman 5 методов: получение списка (в том числе с фильтрацией), получение одной записи, добавление, редактирование, удаление.

**Сделано:**
- Реализовано серверное приложение на Node.js/Express.js с разделением ответственности на уровни: маршрутизации, контроллеров, сервисов и доступа к данным, что обеспечило высокую поддерживаемость и масштабируемость программного кода.
- Разработан полноценный программный интерфейс, поддерживающий базовые методы HTTP (GET, POST, PATCH, DELETE) для управления коллекцией документов, хранящихся в локальном JSON-файле (mock-база данных).
- В слое сервисов реализованы алгоритмы обработки данных, включая автоматическую генерацию уникальных идентификаторов (id) для новых записей и валидацию входящих параметров в контроллерах
- Настроена корректная обработка жизненного цикла HTTP-запроса: от получения параметров из тела или URL до формирования стандартных ответов сервера с соответствующими статус-кодами (например, 201 Created при успешном создании и 204 No Content при удалении)
- Проведено функциональное тестирование разработанных эндпоинтов с помощью Postman, результатом которого стала подготовленная JSON-коллекция запросов, подтверждающая работоспособность всех CRUD-операций.

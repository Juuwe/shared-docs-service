# ЛАБОРАТОРНАЯ РАБОТА №1 (Калькулятор HTML/CSS)

**Задача:** Создание калькулятора. Верстка на HTML, CSS.

**Сделано:**
- С помощью HTML сформирована разметка страницы калькулятора, включающая экран вывода результатов, клавиатурный блок (цифры и операции) и хедер
- Средствами CSS реализовано визуальное оформление элементов с заимствованием цветовой палитры и стилистики сайта Dropbox. Настроено оформление основного контейнера, а также базовые стили кнопок и их интерактивные состояния
- Создано выпадающее меню, появляющееся при наведении на иконку. В него интегрированы ссылка на репозиторий GitHub, селектор для смены темы и раскрывающийся информационный блок об авторе
- Разработаны и интегрированы дополнительные CSS-правила для поддержки альтернативной «темной» темы оформления пользовательского интерфейса.
